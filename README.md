Over the course of 15 tasks, I gained hands-on experience in various domains of Data Science and Machine Learning. The tasks in this repository cover:

* **Data Manipulation & Database querying:** Writing optimized scripts using Python and SQL (e.g., Task 7).

* **Data Visualization:** Building comprehensive visual reports and dashboards (e.g., Ecommerce Sales Dashboard - Task 4).
* **Machine Learning Fundamentals:** Implementing basic ML algorithms, feature engineering, and data preprocessing techniques.
